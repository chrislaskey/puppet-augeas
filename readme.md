About
================================================================================

The `augeas` module installs the required augeas libraries and ruby bindings
for [augeas puppet resource](http://docs.puppetlabs.com/references/latest/type.html#augeas)
on Debian and Redhat derivatives.

```puppet
	class { "augeas": }
```
